# Luck77-Mini-Mart
Lucky 77 Mini Mart - Official Website
Welcome to the official repository for the Lucky 77 Mini Mart digital storefront, located in Bakersfield, CA. This project is designed to bridge the gap between our physical store and our viral social media presence.

🚀 Overview
Lucky 77 Mini Mart is a 24-hour convenience store dedicated to bringing trending TikTok products to our local community. This repository contains the source code for our mobile-first landing page, designed to drive foot traffic and keep customers updated on our latest inventory, such as our popular mystery dumpling squishies.

📦 Features
Mobile-Responsive Design: Optimized for customers browsing on their phones while on the go.

Trending Product Showcase: Highlights viral items like our Kawaii-style dumpling squishies.

Integrated Directions: Direct links to Google Maps for easy navigation to 1134 Niles St.

Store Info: Clearly displays our 24/7 operating status.

🛠 Tech Stack
HTML5/CSS3: Used for a lightweight, fast-loading, and mobile-friendly user experience.

GitHub Pages: Deployed to provide a live, accessible link for social media promotion.
